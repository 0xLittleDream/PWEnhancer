PW Enhancer:

A powerful, lightweight Chrome extension built to fix the most annoying limitations of Physics Wallah (pw.live) and supercharge your study workflow.

No more black screens when taking screenshots, no more manual math to figure out how much time is left on 1.5x speed, and no more guessing how much time you actually spent studying today.

🔥 Key Features
 Slide Captures: Click the new camera icon in the video player to instantly take a clean, high-res screenshot of the lecture slide. Completely bypasses the annoying black-screen DRM.

Auto-Organized Downloads: Screenshots and slides are automatically saved into a Downloads/Physics Wallah/ master folder, neatly organized into perfectly named sub-folders based on the exact lecture topic.

Daily Study Tracker: Automatically tracks every active minute you spend on pw.live. It's smart enough to pause instantly if you switch tabs or minimize the window, ensuring your daily study time is 100% accurate.

TrueTime Calculator: A sleek pill injected into the player that calculates exactly how much real-world time is left in the lecture based on your custom playback speed.

Limitless Custom Speed: Override the native speed limits and push playback up to 4.0x using the popup or the injected + / - hotkeys.

Distraction-Free UI: The player UI gracefully fades away while the video is playing, but instantly locks onto the screen when you pause so you can read the time or grab a screenshot.

🛠️ How to Install (Manual)
Since this extension is not yet on the Chrome Web Store, you can install it manually in a few seconds:
Download the latest .zip release from the Releases page and extract it to a folder.
Open Google Chrome and go to chrome://extensions/.
Turn on Developer mode (toggle switch in the top right corner).
Click the Load unpacked button in the top left.
Select the extracted folder containing the extension files.
Done! Pin the extension to your toolbar and open any PW lecture to see it in action.
